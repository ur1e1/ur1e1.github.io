Iconos CSS

Los iconos se pueden agregar fácilmente a su página HTML mediante el uso de una biblioteca de iconos.

Cómo agregar iconos

La forma más sencilla de agregar un ícono a su página HTML es con una biblioteca de íconos, como Font Awesome. Agregue el nombre de la clase de icono especificada a cualquier elemento HTML en línea (como <i> o <span>). Todos los íconos en las bibliotecas de íconos a continuación son vectores escalables que se pueden personalizar con CSS (tamaño, color, sombra, etc.)


Iconos Awesome Icons

Para usar los íconos de Font Awesome, vaya a fontawesome.com , inicie sesión y obtenga un código para agregar en la <head> sección de su página HTML:

<script src="https://kit.fontawesome.com/yourcode.js" crossorigin="anonymous"></script>

Obtenga más información sobre cómo comenzar con Font Awesome en nuestro tutorial de Font Awesome 5.


Dimensionamiento

De forma predeterminada, los íconos heredan el font-sizede su contenedor principal, lo que les permite hacer coincidir cualquier texto que pueda usar con ellos. Además de eso, Font Awesome viene con las siguientes herramientas de tamaño.

Tamaño relativo

Font Awesome incluye una variedad de tamaños basados ​​en camisetas que no solo aumentan o disminuyen el tamaño de un ícono, sino que también ayudan a alinear verticalmente un ícono con el texto y los elementos que lo rodean (por ejemplo, relleno de botones). Esto es excelente para aumentar/disminuir el tamaño de un ícono mientras se mantiene la legibilidad y la alineación junto con los elementos de la interfaz de usuario.

Animar

Cubriremos los conceptos básicos de las animaciones como el ritmo, el desvanecimiento, el ritmo y el desvanecimiento, la inversión y el giro de sus íconos, para que pueda crear más interés visual en su sitio.

Derrotar

Utilice la fa-beatanimación para escalar un icono hacia arriba o hacia abajo. Esto es útil para llamar la atención o para usar con iconos de salud/centrados en el corazon

Desteñir

Use la fa-fadeanimación para hacer aparecer y desaparecer visualmente un ícono para llamar la atención de una manera sutil (o no tan sutil).

Beat-Fade

Use la fa-beat-fadeanimación para llamar la atención al escalar visualmente y pulsar un ícono hacia adentro y hacia afuera.

Bote

Utilice la fa-bounceanimación para llamar la atención haciendo rebotar visualmente un icono hacia arriba y hacia abajo.

Dar la vuelta

Use la fa-flipanimación para rotar un ícono en el espacio 3D (abre en ventana nueva). De forma predeterminada, voltear gira un icono sobre el eje Y 180 grados. Voltear es útil para transiciones, estados de procesamiento o para usar objetos físicos que uno voltea en el mundo real.

Sacudir

Use la fa-shakeanimación para llamar la atención o notar que algo no está permitido agitando un icono de un lado a otro.

Girar

Use la fa-spinclase para hacer que cualquier ícono gire, y use fa-spin-pulsepara que gire con ocho pasos. Esto funciona especialmente bien con fa-spinnery todo en la categoría de iconos giratorios . Si desea que un icono gire al revés (por ejemplo, en sentido contrario a las agujas del reloj), agregue una clase adicional de fa-spin-reverse.

Pila

Para apilar varios íconos, use la fa-stackclase en el elemento HTML principal de los 2 íconos que desea apilar. Luego agregue la fa-stack-1xclase para el ícono de tamaño regular y agregue la fa-stack-2xclase para el ícono más grande. fa-inversese puede agregar al ícono con fa-stack-1xpara ayudar con un efecto de apariencia deslumbrante. Incluso puede lanzar clases de iconos más grandes en el padre para obtener un mayor control del tamaño.

Alineación de iconos apilados y regulares

Puede usar íconos apilados junto con íconos individuales. Dado que los íconos apilados están diseñados por el CSS incluido en nuestro marco para que tengan el doble del tamaño de los íconos individuales, deberá reducir el tamaño del ícono apilado o aumentar el tamaño de los íconos individuales.

Cambiar el tamaño de fuente predeterminado con apilamiento

Cuando apila íconos pero cambia el tamaño de fuente, deberá realizar algunos ajustes en su CSS para mantener los íconos alineados.

Iconos en una lista

Cubriremos los conceptos básicos de voltear íconos, escalarlos hacia arriba o hacia abajo y moverlos hacia la izquierda o hacia la derecha, para obtener algunos efectos súper útiles.

Escalada

El escalado de Power Transform afecta el tamaño del ícono sin cambiar o mover el contenedor. Para escalar iconos hacia arriba o hacia abajo, use grow-#y shrink-#con cualquier valor arbitrario, incluidos los decimales. Las unidades son 1/16em. Para mayor claridad en el ejemplo, hemos agregado un color de fondo en el ícono para que pueda ver el efecto.

Posicionamiento

El posicionamiento de Power Transform afecta la ubicación del ícono sin cambiar o mover el contenedor. Para mover iconos hacia arriba, abajo, izquierda o derecha, use up-#, down-#, left-#y right-#con cualquier valor arbitrario, incluidos los decimales. Las unidades son 1/16em. Para mayor claridad en el ejemplo, hemos agregado un color de fondo en el ícono para que pueda ver el efecto.

Girar y voltear

La rotación y volteo de Power Transform afecta el ángulo y el reflejo del ícono sin cambiar o mover el contenedor. Para rotar o voltear íconos, use cualquier combinación de rotate-#, flip-vy flip-hcon cualquier valor arbitrario. Las unidades son grados con números negativos permitidos (vea el quinto ícono en el ejemplo). Para mayor claridad en el ejemplo, hemos agregado un color de fondo en el ícono para que pueda ver el efecto.
